ubuntu使用shadowsocks

Written by hwholiday with ♥ on 7 April 2017 in Linux 1 min

安装shadowsocks

用PIP安装
sudo apt-get update
sudo apt-get install python-pip
sudo apt-get install python-setuptools m2crypto
pip install shadowsocks
如果是ubuntu16.04 直接 (16.04 里可以直接用apt 而不用 apt-get 这是一项改进）
sudo apt install shadowsocks

shadowsock使用

推荐在/etc/shadowsocks下  新建config.json

{
    "server":"118.193.178.111",//你的服务器IP
    "server_port":60010,//服务器端口
    "local_address": "127.0.0.1",//本地代理IP
    "local_port":1080,//本地代理端口
    "password":"123qeq",//密码
    "timeout":300,//过期时间
    "method":"aes-256-cfb",//加密方式
    "fast_open": false, //默认
    "workers": 1//默认
}

使用命令:nohup sslocal -c /etc/shadowsocks/config.json >/dev/null 2>&1 & 启动代理

如果使用supervosor控制程序可参考如下配置

[program:shadowsocks]
command=sslocal -c /etc/shadowsocks.json
autorestart=true
user=nobody

google浏览器配置

  • 下载 https://github.com/FelisCatus/SwitchyOmega/releases 后缀为crx
  • 打开chrome的设置>扩展程序，然后把插件拖进来安装
  • 切换到switchyOmega主界面 > 新建情景模式 > 选择代理服务器 > 名称自己定义(shadowcocks) > 创建 > 应用选项
  • 选择新建的情景模式 > 代理协议选择为SOCKS5 > 代理服务器输入127.0.1 > 代理端口输入1080,这些配置都是从/etc/shadowsocks/config.json拿出来的
  • 选择auto switch >把规则列表规则的情景模式选择为(shadowcocks) > 规则列表设置 > 规则列表格式选择为AutoProxy > 规则列表网址输入(https://raw.githubusercontent.com/gfwlist/gfwlist/master/gfwlist.txt) > 点击立即更新情景模式
  • 返回浏览器主页点击SwitchyOmega图标 选择 auto switch
  • 访问www.google.com 来测试下你的服务吧
